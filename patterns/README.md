# Golden Code Examples

## Selection Criteria
- Represents common patterns (not edge cases)
- High quality (9-10/10 rating)
- Consistent style within type
- Well-documented

## Pattern Types
- handlers/ - HTTP request handlers
- services/ - Business logic layer
- middleware/ - HTTP middleware
- models/ - Data models

## Adding Patterns
1. Copy perfect example to appropriate directory
2. Document why it's perfect
3. Reindex: `python scripts/index_patterns.py`
